# 示例 Prompt - 调用 Industry Research Suite V2

## 完整工作流调用

```
请对「智能穿戴设备」行业进行完整调研，输出产品迭代战略规划书。
```

### 调用市场情报专员
```
请搜索「智能穿戴设备」行业的最新动态和竞品消息，输出情报简报。
```

### 调用用户研究专家
```
请分析「智能手表」产品的用户痛点和体验问题，输出洞察报告。
```

### 调用商业分析师
```
请对「TWS耳机」市场进行量化分析，评估风险并给出建议。
```

### 调用产品战略官
```
请制定「智能家居」产品的战略规划，包含SWOT和路线图。
```

## 快速调研模式

```
请快速调研「新能源汽车」行业，输出1-2页摘要。
```
